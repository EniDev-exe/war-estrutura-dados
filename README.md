# Desafio War Estruturado

Projeto do jogo War para a disciplina de Estrutura de Dados.

## Objetivo
Implementar conceitos básicos de estrutura de dados em um jogo War simplificado.

##  Estruturas Utilizadas
- **Array estático** para territórios
- **Struct** para organizar dados do jogo
- **Funções modularizadas** para cada operação

## Como Compilar
```bash
gcc main.c war_functions.c -o war_game
